This library provides high-\/res time and sleep functions, as well as unit conversions functions.

Even though {\ttfamily ptime\+\_\+t} is expressed in microseconds, it is still recommended that you use the {\ttfamily pt\+\_\+to\+\_\+usec} and {\ttfamily pt\+\_\+from\+\_\+usec} functions should this ever change.\hypertarget{pico__time_8h_autotoc_md56}{}\doxysubsubsection{Usage\+:}\label{pico__time_8h_autotoc_md56}
To use this library in your project, add the following

\begin{quote}
\#define PICO\+\_\+\+TIME\+\_\+\+IMPLEMENTATION \#include \char`\"{}pico\+\_\+time.\+h\char`\"{} \end{quote}
to a source file (once).

IMPORTANT\+: On POSIX systems, when defining PICO\+\_\+\+TIME\+\_\+\+IMPLEMENTATION, one of three conditions must hold\+:

1) \#define \+\_\+\+POSIX\+\_\+\+C\+\_\+\+SOURCE 199309L must precede any header include in the file 2) This library must be included before any other headers 3) The build system must define {\ttfamily \+\_\+\+POSIX\+\_\+\+C\+\_\+\+SOURCE 199309L}
