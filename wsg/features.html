Features currently supported by the engine:

  • Graphics (sprites/animations) using custom 2D renderer (with batching draw calls for performance)
  • Music and basic 2D positional audio
  • Basic 2D physics (linear and rotational kinematics, but only linear dynamics). This physics engine can be swapped out for a more powerful one if needed.
  • High-performance, multi-threaded architecture
  • ECS (Entity-Component-System) framework for rapid development of game logic
  • Particle systems (designed, but not implemented yet)
  • Basic AI support (finite state machines)
  • Tool support: PhysicsEditor, TexturePacker, and Tiled editor (tile map loading and rendering)
  • Platform support via SDL for PC (Window, Linux, MacOS), mobile (Android, iOS), and possibly consoles (PS5, Xbox, and others...)
